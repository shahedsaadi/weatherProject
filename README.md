### Overview

Weather web application, user can write any city to know the weather description and temperature of that city. So the weather data dynamic based on what the user typed.

#### Technology Stack:

- HTML5.
- Node.js.
- JavaScript.
- Expressjs.
- APIs.

#### To run the project:

- Run npm install.
- Run node app.js.
- Visit the link http://localhost:3000/
